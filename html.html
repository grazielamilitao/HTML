HTML

O HTML sem dúvida é uma das linguagens mais importantes para programação WEB, por que é ela que exibe a informação. Além de exibir a informação, ela dá significado. Isso é importante por que alguns sistemas como o Google, que irão ler sua página, precisam entender o que é cada elemento nela e o que cada um desses elementos significam. O acrônico HTML significa: HyperText Markup Language, que em português fica: Linguagem de Marcação de Hipertexto. Por trás das palavras Hipertexto e Marcação tem muita história e guardam a real essência da função do HTML. Semântica é a organização da informação usando HTML. O principal a se saber sobre HTML é: HTML serve para dar significado e organizar a informação dos websites.


Composição do HTML

Um documento HTML é composto por tags, as quais possuem um nome e aparecem entre os sinais < e >, por exemplo, em < html > e < head >. As tags precisam ser abertas e fechadas, como em < body > < / body >. Nesse caso, a tag de fechamento deve conter a barra / antes do nome. Outras, porém, não precisam ser fechadas, como a tag < meta >. Nesses casos, a adição da barra / no final da própria tag é opcional. Outro aspecto importante da linguagem é que ela é case insensitive, ou seja, não leva em consideração a diferença entre letras maiúsculas e minúsculas. No entanto, o uso apenas de letras minúsculas tem sido utilizado como padrão pelos desenvolvedores. Além dessas características, as tags também possuem atributos na qual a tag meta possui o atributo charset = "UTF-8". Essas propriedades definem algumas características adicionais de cada tag e em alguns casos são obrigatórias. Seus valores devem aparecer entre aspas duplas, como no exemplo acima, ou em aspas simples, caso o próprio valor contenha aspas duplas.
