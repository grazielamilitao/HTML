JavaScript

O JavaScript (JS) é uma linguagem de programação utilizada principalmente em páginas web. Com o JS, pode-se mostrar mensagens e outras informações interessantes, fazer verificações ou mudar dinamicamente a apresentação visual das páginas, conforme o comportamento desejado que a página (ou aplicação) possua, tudo através do navegador do usuário. É possível fazer isso até mesmo sem conexão com a internet.

Por que devo usar o JavaScript?

Atualmente, o JavaScript é a linguagem web mais popular do mundo, pois todos os sites modernos e a grande maioria das aplicações web possuem recursos em JavaScript, como por exemplo: galeria de imagem, interação com a rolagem da página, animações parallax e validações e manipulação de elementos HTML e CSS. Hoje em dia, até banco de dados NoSQL podem ser baseados em JavaScript. Essa linguagem é essencial para qualquer programador e é simples de aprender. Trata-se de uma linguagem baseada em eventos. E ainda conta com diversos frameworks, como:

  1. AngularJS
  2. Backbone.js
  3. React.JS
  4. Ember.js
  5. Node.Js

Outras aplicações do JavaScript:

  • Validação de formulários;
  • Esconder ou apresentar informações da página;
  • Janelas de aviso ou de confirmação;
  • Sistemas de autocomplete;
  • Aplicações web que mostram informações que mudam constantemente;
  • Interações com elementos HTML;
  • Interação com regras CSS.

Comandos do Console JavaScript

Comando Descrição Exemplo
assert(expression, message) Envia uma mensagem se expression for avaliada como false. console.assert((x == 1), "assert message: x != 1");
clear() Limpa as mensagens da janela do console. console.clear();
count(title) Envia o número de vezes que o comando count foi chamado para a janela do console. console.count();
console.count("inner loop");
debug(message) Envia message para a janela do console. console.debug("logging message");
dir(object) Envia o objeto especificado para a janela do console e o exibe em um visualizador de objetos. console.dir(obj);
error(message) Envia message para a janela do console. console.error("error message");
